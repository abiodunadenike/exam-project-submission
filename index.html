A BRIEF DESCRIPTION OF MY PROJECT

My project is about deploying a static website with AWS EC2 using Nginx.
Logged into AWS, created an EC2 instance step by step by choosing a name for my instance, choose a base operating system, select an instance type create a keypair and give it a name, set rules through security groups ensure the following rules are incuded: HTTP(port 80), HTTPS(port 443) and SSH(port 22) review and launch, verify all configurations and launch instance. Connect SSH into the instance using the keypair and the actual ip address to connect, then install Nginx, start and enable the Nginx service make sure it is active and running. Also created a simple HTML page with neccessary infomation needed, then use scp tool to extract and move to Nginx's web directory deploy to the server.
